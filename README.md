# 🍬 HANAPIN MO ANG KAPAREHAS KO BEYBE!

A classic Match-3 puzzle game built with HTML, CSS, and JavaScript.

## Features
- 8x8 game grid with 6 unique tile types
- Swap adjacent tiles to create matches of 3 or more
- Chain reaction system for cascading matches
- Score multipliers for longer chains
- Win condition: Reach 1200 points
- Loss condition: No valid moves remaining
- Responsive design for mobile and desktop
- Start menu and back-to-menu functionality

## How to Play
1. Press ENTER or click PLAY to start
2. Click on a tile to select it
3. Click on an adjacent tile to swap
4. Create horizontal or vertical lines of 3+ matching tiles
5. Watch for chain reactions that create bonus points!
6. Don't let the moves out of

## Scoring
- Base: 10 points per matched tile
- Chain multipliers:
  - Chain 1: 1x
  - Chain 2: 1.5x
  - Chain 3: 2x
  - Chain 4+: 2.5x
